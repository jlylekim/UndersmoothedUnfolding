UndersmoothedUnfolding¶

UndersmoothedUnfolding provides a ROOT implementation of an unfolding technique that aims at optimal uncertainty quantification for an unfolded particle spectrum. The implementation is an extension of the TUnfold library and can hence be easily incorporated into existing TUnfold-based particle physics data analyses. This website provides the documentation for using and installing UndersmoothedUnfolding which can be downloaded by cloning this Git repository.
